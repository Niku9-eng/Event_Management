Event Management System (Flask)
📌 Project Description

The Event Management System is a web application developed using Flask (Python) that allows users to view and book seats for events while providing administrators with the ability to manage events through an admin dashboard.

The system implements authentication, role-based access control, and CRUD operations for event management.

🚀 Features
👤 User

Register a new account

Login securely

View available events

Book seats for events

Access only user-specific features

🛠 Admin

Secure admin login

Add new events

View all events

Manage events through admin dashboard

Role-based access (only admin can add events)
